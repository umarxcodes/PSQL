# PostgreSQL Notes and Guide

This repository contains a personal PostgreSQL reference and learning guide.
It is centered around the `notes.md` document, which covers PostgreSQL concepts from beginner to senior engineer level.

## What’s in this repo

- `psql/notes.md` — a comprehensive guide to PostgreSQL, including:
  - installation and setup
  - core database concepts
  - data types and table design
  - CRUD operations
  - indexing, performance, and optimization
  - transactions, views, functions, and triggers
  - JSON/JSONB support, window functions, security, backup, and more

## How to use it

1. Open `psql/notes.md` in your editor.
2. Read through the sections in order or jump to the topics you need.
3. Use the examples as a PostgreSQL study reference or quick cheat sheet.

## Quick start

```bash
cd /home/engineer/projects/psql
ls psql
# open the notes file
less psql/notes.md
```

## Purpose

This repo is a learning and reference project for PostgreSQL concepts, best practices, and real-world database design.
It can be used as a study guide or quick reference while working with PostgreSQL.
